# Personal Web CV

Welcome to my personal web-based CV, which can be viewed [here](https://dankolman.cz/cv/).

## Overview

This project represents my online résumé, designed to present my professional experience, skills, and background in an interactive and responsive format. The website was developed using HTML, CSS, and JavaScript, ensuring a clean and modern design that adapts seamlessly to both desktop and mobile devices.

## Features

- **Responsive Design**: The layout adjusts to various screen sizes for optimal viewing on desktop, tablet, and mobile devices.
- **Last Updated**: The website dynamically displays the last time the page was modified using JavaScript.
- **Simple and Clean Layout**: The structure is minimalist, providing easy navigation and quick access to key information.

## Technologies Used

- HTML5
- CSS3 (with media queries for responsiveness)
- JavaScript for lasta page update
